Bem-vindo ao micro-site'107 !

Micro-site do trabalho P3 de MAS pelo grupo 107

Versão atualizada

Este projeto considera a possibilidade de colocar à porta de cada farmácia um quiosque com uma interface que permita que o utente compre um medicamento não sujeito a receita médica. Esta interface pretende tanto facilitar a tarefa de comprar um medicamento às pessoas como ajudar os farmacêuticos em dar vazão a filas.

Todas as máquinas devem ser controladas por um sistema central, para o qual as várias máquinas enviam um relatório periodicamente (por exemplo, com informação dos medicamentos mais procurados) e avisos (por exemplo, de quando o stock de um medicamento está prestes a acabar) quando necessário.
